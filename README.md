# Инструкция по запуску проекта

## Сервер (Backend)

1. Перейдите в директорию `server`:
    ```bash
    cd server
    ```
2. Установите необходимые зависимости:
    ```bash
    npm install ws
    ```
3. Запустите сервер:
    ```bash
    node server
    ```

---

## Клиент (Client)

1. Перейдите в директорию `client`:
    ```bash
    cd client
    ```
2. Установите зависимости:
    ```bash
    npm install
    ```
3. Запустите клиентское приложение:
    ```bash
    npm start
    ```

---
